pybedtools.bedtool.BedTool.file_type¶

BedTool.file_type¶

Return the type of the current file. One of (‘bed’,’vcf’,’gff’, ‘bam’, ‘sam’, ‘empty’).

>>> a = pybedtools.example_bedtool('a.bed')
>>> print(a.file_type)
bed
